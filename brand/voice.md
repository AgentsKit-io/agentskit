# AgentsKit ecosystem — voice & canonical descriptions

Single source for how every property describes itself. Cross-link anchor text,
ecosystem-bar labels, OG copy, and meta descriptions draw **only** from here.
Never assert a feature a property does not claim about itself.

## Canonical one-liners (use verbatim)

| Property | One-liner | What it IS — and is NOT |
|----------|-----------|--------------------------|
| AgentsKit (`agentskit.io`) | **The most complete agent toolkit for JavaScript.** | Open-source library/SDK to build agents in code. |
| AgentsKit OS (`akos.agentskit.io`) | **The operating system for AI agents in production.** | A complete product — governed flows, sandbox, egress allow-list, signed audit, cost-metering; runs as managed cloud, self-hosted desktop, or terminal. **NOT** a deploy button or a hosting add-on for the library. |
| Agents Playbook (`playbook.agentskit.io`) | **Best practices for building production AI agents.** | Methodology — pillars, patterns, gate scripts. Guidance, not code. |
| Registry (`registry.agentskit.io`) | **Ready-to-use AI agents.** | shadcn-style installable agents (`npx agentskit add`). |

## Positioning sentence (shared)

> Grab what you want from **AgentsKit**, follow best practices in the
> **Playbook**, drop in ready-made agents from the **Registry**, and run them in
> production on **AKOS** — the operating system for AI agents in production.

## Rules

- **No invention.** An anchor toward AKOS positions it as *the production OS for
  AI agents* — it does not claim AKOS sub-features (scheduler, trace viewer, SSO,
  edge, "deploy your agent here") unless verified in AKOS's own docs. Same for
  every property: concise, accurate, no fabrication.
- **Relationship, stated plainly:** AgentsKit = the library you build with; AKOS
  = the production OS, built on it; Playbook = how to do it well; Registry = pre-
  built agents to start from.
- **No personal attribution on product surfaces.** `emersonbraun.dev` links out
  to the four; the four do not link back with "built by" credits.
- **Numbers come from the stats manifest** ([[ecosystem-stats]]), never hand-typed.
