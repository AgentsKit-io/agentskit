---
title: upstashVector
description: Serverless HTTP vector DB from Upstash.
---

```ts
import { upstashVector } from '@agentskit/memory'

const store = upstashVector({
  url: process.env.UPSTASH_VECTOR_REST_URL!,
  token: process.env.UPSTASH_VECTOR_REST_TOKEN!,
  namespace: 'prod',
})
```

## Options

| Option | Type |
|---|---|
| `url` | `string` |
| `token` | `string` |
| `namespace` | `string?` |
