## A 3-step recipe for production-ready system prompts

1. **Define explicit boundaries.** List what the agent must avoid (e.g., "Do not discuss X unless Y").
2. **Embed role clarity.** Specify the agent's purpose and constraints (e.g., "You are a code assistant — only output code").
3. **Test with edge cases.** Throw boundary-pushing queries at your configured harness to validate the guardrails hold.
